基于以太坊区块链应用集成开发环境

本项目为东北大学大学生创新创业训练计划资助项目（X20191014500186），受中央高校基本科研业务专项资金资助（N182410001）。

基本情况

  • 校级编码：X20191014500186
  • 立项编码：190227
  • 项目名称：基于以太坊区块链应用集成开发环境
  • 申请团队情况：核心成员：2人；预备成员：3人
  • 指导教师：1人，副教授
  • 项目来源：教师子课题
  • 项目类型：资助
  • 学科类别：信息技术-计算机
  • 申请经费：15000元（资助5000元）
  • 起止时间：2018年12月-2020年5月

立项背景

研究内容

  1. 区块链集成开发环境界面与功能设计。
  2. 以太坊分层模块库的实现。下图中列举了需要使用C++、Go编程语言进行实现的部分库：
  3. 基于现有框架（如DApp）的以太坊区块链应用开发支持。
  4. 整合以太坊应用相关的调试工具。

研究意义

本项目致力于设计并开发出一款基于以太坊的区块链应用集成开发环境，它能够综合现有以太坊开发工具的优点，同时支持以太坊区块链应用、智能合约、开发框架等相关的研究与开发，并且后期也将会不断加入对其他主流区块链平台相关开发框架的支持。

鉴于现在尚不存在一款较为完善的以太坊集成开发环境，如果能够成功开发出这样一款成长型集成开发环境，对于区块链开发领域应该是一个开创性的成果，有利于区块链开发产业的蓬勃发展。

项目的特色与创新之处

  1. 理论创新：通过本项目深入研究面向区块链技术的软件开发框架理论，为区块链开发领域提供了理论创新。
  2. 方法新颖：实施层面采用了层次化设计的思想，按照区块链六层架构逐层、逐模块进行开发设计，环环紧扣，层层递进。
  3. 应用广泛：本项目的研究成果（集成开发环境）可以用于不同领域区块链系统便捷开发，以及自定义个性化系统的实现。

实施方案

略。

项目研究预期成果

  1. 设计并开发出一款基于以太坊区块链的应用集成开发环境软件以及相关的模块库，并以此申请相关的软件著作权和专利。
  2. 发表介绍项目成果、运用该集成开发环境进行开发或实验的相关论文。

拟利用资源

  1. 设备：项目所需设备主要由实验室现有的台式计算机、项目组成员的笔记本电脑以及项目进行过程中拟采购的台式计算机和服务器等组成。计算机主要用于设计、开发以及文档的编写，服务器用于测试编写的区块链项目。
  2. 场地：项目所需场地为实验室现有的工位以及即将向学校、学院申请的工位等。这些工位用于项目组日常研究、讨论使用。
  3. 其他资源：项目所需软件和库主要采用开源、非商业的，或者面向学生的版本，项目进行中可能有其他需要采购的正版软件和库。这些软件和库包括但不限于：
    1. JetBrains公司的CLion软件：用于C或C++程序开发；
    2. Microsoft公司的Office套件：用于文档、表格和演示文稿的制作；
    3. Qt公司的Qt库：用于C++编写的软件跨平台界面开发；
    4. Google公司的Go编译工具：用于支持区块链的构建；
    5. Microsoft公司的Windows操作系统、Canonical公司的Ubuntu操作系统和Apple公司的MacOS操作系统。

经费使用计划

  1. 办公费（图书、打印等）：1200元
  2. 实验材料费：6400元
  3. 测试、化验、加工费：3000元
  4. 外出调研费用：1000元
  5. 论文版面费：2400元
  6. 专利申请费：1000元
